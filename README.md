# University Enrollment System

## Project Description
This project is a console-based University Enrollment System developed using C++.
It allows administrators to manage students, courses, and enrollments efficiently.

## Features
- Add Student 
- View Student List
- Add Course
- Enroll in Course
- Withdraw from Course
- View Course Roster
- User Login & Logout System
- Menu-driven interface

2. Build and Run the program.
